# React component — Holberton School

This folder contains the **React component** project (class components, state, and lifecycle).

Each task lives in its own directory (`task_0`, `task_1`, …) with a Vite dashboard app under `dashboard/`.

## Environment

- Ubuntu 20.04 LTS
- Node.js 20.x
- npm 10.x
- Jest 29.7.0 (global install: `npm install -g jest@29.7.0`)
