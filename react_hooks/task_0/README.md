# React Hooks — Task 0

## Objective

Modernize the React codebase by replacing class components and lifecycle usage with **functional components** and **React hooks**.

## Requirements covered

- `Header` is a **functional component**
- Context is consumed with **`useContext`**
- Same JSX structure is preserved
- When the user is logged in, a welcome message and a logout link are displayed
- Clicking the logout link calls `logOut`
